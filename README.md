# Missao STEAM CEEJA

Game educativo para formacao pedagogica de CEEJA sobre Educacao STEAM e ABP.

## Como jogar localmente

Abra `index.html` no navegador ou publique a pasta em um host estatico.

## Publicar no Vercel

1. Crie um repositorio no GitHub.
2. Envie todos os arquivos desta pasta. Deixe `infografico-steam.png` junto de `index.html`, `app.js`, `styles.css` e `app-config.js`.
3. No Vercel, importe o repositorio.
4. Framework preset: `Other`.
5. Build command: deixe vazio.
6. Output directory: deixe vazio ou use `.`.

## Ranking online com Supabase

Sem configuracao, o ranking funciona localmente no navegador. Para ranking compartilhado entre jogadores, crie um projeto gratuito no Supabase e rode este SQL no SQL Editor:

```sql
create table if not exists public.steam_scores (
  id bigint generated by default as identity primary key,
  name text not null check (char_length(name) <= 28),
  score integer not null check (score >= 0),
  created_at timestamptz not null default now()
);

alter table public.steam_scores enable row level security;

create policy "Qualquer pessoa pode ver o ranking"
on public.steam_scores
for select
using (true);

create policy "Qualquer pessoa pode inserir pontuacao"
on public.steam_scores
for insert
with check (true);
```

Depois, edite `app-config.js`:

```js
window.STEAM_GAME_CONFIG = {
  SUPABASE_URL: "https://SEU-PROJETO.supabase.co",
  SUPABASE_ANON_KEY: "SUA-ANON-PUBLIC-KEY"
};
```

A chave `anon public` do Supabase e propria para uso no navegador quando as politicas RLS estao configuradas.

## Personalizacao rapida

- Perguntas e respostas: edite o array `challenges` em `app.js`.
- Quantidade de rodadas: altere `TOTAL_ROUNDS` em `app.js`.
- Cores e layout: edite `styles.css`.
